Create a resume

  • I've provided you a starting HTML template and an example resume.
  • Save the HTML template to your machine, and edit it to create a resume like the example. You should enter your own information in the resume. Your information doesn't have to be perfect, but make your best effort; you can update and correct the dates and titles on your own. If you are completely at a loss for content, you can make things up or use the example as a guide.
  • Guidelines:
    • Name your file FullName-resume.html
    • Keep your code in one document. Your CSS should be embedded on the page (in the <head> section).
    • Don't use tables: you don't need them. Hint: imagine blocking the content with boxes. Some of your boxes will need to float, and some will need to be given explicit widths.
    • Use proper indentation (see HTML/CSS technology lecture notes)
    • Your markup must be valid
    • Use the web, e.g. W3Schools (html, css), for reference.
    • You will be graded based on the the Lab 2 rubric.
  • We will work through some of this together in class.
  • Feel free to experiment and be creative - but show us that you can create something similar to the example resume.
